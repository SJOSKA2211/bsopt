║                  CRANK-NICOLSON FINITE DIFFERENCE SOLVER                       ║
║                        Mathematical Architecture                               ║
╚════════════════════════════════════════════════════════════════════════════════╝

┌────────────────────────────────────────────────────────────────────────────────┐
│ INPUT PARAMETERS                                                               │
├────────────────────────────────────────────────────────────────────────────────┤
│  Market Data:          Grid Configuration:                                     │
│  • Spot (S₀)           • n_spots: M+1 spatial points                          │
│  • Strike (K)          • n_time:  N temporal steps                            │
│  • Maturity (T)        • S_min = 0.01                                         │
│  • Volatility (σ)      • S_max = 3×K                                          │
│  • Rate (r)            • dS = (S_max - S_min) / M                             │
│  • Dividend (q)        • dt = T / N                                           │
│  • Option type         • Mesh ratio: r = dt/dS²                               │
└────────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌────────────────────────────────────────────────────────────────────────────────┐
│ GRID CONSTRUCTION                                                              │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  Spatial Grid (Uniform):                                                       │
│  ┌───────────────────────────────────────────────────────────────┐            │
│  │ S₀=0.01   S₁    S₂   ...   Sᵢ   ...   S_{M-1}   S_M=3K       │            │
│  └───┬───────┬─────┬─────────┬─────────┬──────────┬──────┘      │            │
│      │       │     │         │         │          │                           │
│      └───────┴─────┴─────────┴─────────┴──────────┴──────────────►            │
│      0      dS    2dS       i×dS                              S                │
│                                                                                │
│  Temporal Grid (Backward):                                                     │
│  ┌───────────────────────────────────────────────────────────────┐            │
│  │ t=0   t₁   t₂   ...   tₙ   ...   t_{N-1}   t_N=T             │            │
│  └───┬───────┬────┬────────┬────────┬──────────┬───────┘         │            │
│      │       │    │        │        │          │                              │
│      └───────┴────┴────────┴────────┴──────────┴─────────────────►            │
│  PRESENT                                             MATURITY      t           │
│  (solve backward from T to 0)                                                  │
└────────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌────────────────────────────────────────────────────────────────────────────────┐
│ BLACK-SCHOLES PDE                                                              │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  ∂V     1  2  2 ∂²V              ∂V                                            │
│  ── + ─ σ  S  ──── + (r-q)S ─── - rV = 0                                      │
│  ∂t    2        ∂S²              ∂S                                            │
│                                                                                │
│  Subject to:                                                                   │
│  • Terminal: V(S,T) = max(S-K, 0)  [call] or max(K-S, 0)  [put]               │
│  • Lower:    V(0,t) = 0 [call]    or K×e^(-r(T-t)) [put]                      │
│  • Upper:    V(∞,t) = S-K×e^(-r(T-t)) [call] or 0 [put]                       │
└────────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌────────────────────────────────────────────────────────────────────────────────┐
│ DISCRETIZATION: Finite Differences                                             │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  Spatial (Central Differences):                                                │
│  ┌──────────────────────────────────────────────────────────────┐             │
│  │ ∂V     V_{i+1} - V_{i-1}                                     │             │
│  │ ── ≈ ──────────────────    [O(dS²) accuracy]                │             │
│  │ ∂S          2dS                                              │             │
│  │                                                              │             │
│  │ ∂²V    V_{i+1} - 2V_i + V_{i-1}                              │             │
│  │ ─── ≈ ────────────────────────   [O(dS²) accuracy]          │             │
│  │ ∂S²            (dS)²                                         │             │
│  └──────────────────────────────────────────────────────────────┘             │
│                                                                                │
│  Temporal (Crank-Nicolson = Trapezoidal Rule):                                 │
│  ┌──────────────────────────────────────────────────────────────┐             │
│  │ V^{n-1}_i - V^n_i                                            │             │
│  │ ───────────────── = 0.5 × [L(V^{n-1}) + L(V^n)]             │             │
│  │       dt                                                     │             │
│  │                                                              │             │
│  │ where L is the Black-Scholes spatial operator                │             │
│  │                                      [O(dt²) accuracy]       │             │
│  └──────────────────────────────────────────────────────────────┘             │
└────────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌────────────────────────────────────────────────────────────────────────────────┐
│ COEFFICIENT CALCULATION (at each grid point i)                                 │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  αᵢ = 0.25×dt × [σ²Sᵢ²/dS² - (r-q)Sᵢ/(2dS)]   (lower diagonal)               │
│  βᵢ = -0.5×dt × [σ²Sᵢ²/dS² + r]                (main diagonal)                │
│  γᵢ = 0.25×dt × [σ²Sᵢ²/dS² + (r-q)Sᵢ/(2dS)]   (upper diagonal)               │
│                                                                                │
│  Physical interpretation:                                                      │
│  • α: Weight for downward diffusion and drift                                 │
│  • β: Weight for zero-order term (discounting)                                │
│  • γ: Weight for upward diffusion and drift                                   │
└────────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌────────────────────────────────────────────────────────────────────────────────┐
│ MATRIX CONSTRUCTION (Tridiagonal)                                              │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  Implicit Matrix A:              Explicit Matrix B:                            │
│  ┌                         ┐     ┌                         ┐                  │
│  │  1                      │     │  1                      │                  │
│  │ -α₁  1-β₁  -γ₁          │     │  α₁  1+β₁   γ₁          │                  │
│  │     -α₂  1-β₂  -γ₂      │     │      α₂  1+β₂   γ₂      │                  │
│  │         ...   ...   ... │     │          ...   ...   ... │                  │
│  │            -αₘ 1-βₘ -γₘ │     │             αₘ 1+βₘ  γₘ │                  │
│  │                       1 │     │                       1 │                  │
│  └                         ┘     └                         ┘                  │
│                                                                                │
│  Sparse CSR format: Only 3 diagonals stored (O(M) memory)                     │
│  First/last rows = identity (enforce Dirichlet boundaries)                    │
└────────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌────────────────────────────────────────────────────────────────────────────────┐
│ TIME-STEPPING ALGORITHM (Backward from T to 0)                                 │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  Initialize: V^N = payoff(S)                                                   │
│              ┌───────────────────────────────┐                                │
│              │ V^N_i = max(S_i - K, 0)       │  [call]                        │
│              │      or max(K - S_i, 0)       │  [put]                         │
│              └───────────────────────────────┘                                │
│                                                                                │
│  FOR n = N, N-1, ..., 2, 1:                                                    │
│    ┌───────────────────────────────────────────┐                              │
│    │ 1. RHS: b = B × V^n                       │  [Matrix-vector product]     │
│    │                                           │                              │
│    │ 2. Enforce boundaries:                    │                              │
│    │    b[0] = lower_boundary(t_n)             │                              │
│    │    b[M] = upper_boundary(t_n)             │                              │
│    │                                           │                              │
│    │ 3. Solve: A × V^{n-1} = b                 │  [Sparse linear system]      │
│    │                                           │  [Thomas algorithm O(M)]     │
│    │                                           │                              │
│    │ 4. Update: V ← V^{n-1}                    │                              │
│    └───────────────────────────────────────────┘                              │
│                                                                                │
│  Result: V^0 (option values at present time)                                   │
└────────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌────────────────────────────────────────────────────────────────────────────────┐
│ INTERPOLATION (Find V(S₀) from grid)                                          │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  Linear interpolation between grid points:                                     │
│  ┌──────────────────────────────────────────────────────────────┐             │
│  │ If S_i ≤ S₀ ≤ S_{i+1}, then:                                │             │
│  │                                                              │             │
│  │         S₀ - S_i                                             │             │
│  │ V(S₀) = ──────── × (V_{i+1} - V_i) + V_i                    │             │
│  │        S_{i+1} - S_i                                         │             │
│  └──────────────────────────────────────────────────────────────┘             │
│                                                                                │
│  ──────●─────────────●─────────────●────────                                  │
│        S_i          S₀            S_{i+1}                                      │
│        V_i        V(S₀)           V_{i+1}                                      │
└────────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌────────────────────────────────────────────────────────────────────────────────┐
│ OUTPUT: OPTION PRICE                                                           │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  V(S₀, t=0) = Option price at current spot                                    │
│                                                                                │
│  Accuracy: O(dt² + dS²)                                                        │
│  Stability: Unconditionally stable                                             │
└────────────────────────────────────────────────────────────────────────────────┘

├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  Delta:  Δ = [V(S₀+h) - V(S₀-h)] / (2h)                                       │

│                                                                                │
│  ───────●───────●───────●───────  (use grid solution)                          │
│       S₀-h     S₀     S₀+h                                                     │
│                                                                                │
│  Cost: O(1) - No additional PDE solve required                                │
└────────────────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────────────┐
│ VEGA, THETA, RHO (Bump and recompute)                                          │
├────────────────────────────────────────────────────────────────────────────────┤
│                                                                                │
│  Vega:  ν = [V(σ+ε) - V(σ-ε)] / (2ε)        ε = 0.01                          │
│         ┌─────────────────┐  ┌─────────────────┐                              │
│         │ Solve with σ+ε  │  │ Solve with σ-ε  │                              │

│                  └────────┬───────────┘                                        │

│         │ Solve with T-δt │                                                   │
│         └─────────────────┘                                                   │

│         │ Solve with r+ε  │  │ Solve with r-ε  │                              │

│                  │                    │                                        │

╚════════════════════════════════════════════════════════════════════════════════╝

  Von Neumann Stability Analysis:
  ┌──────────────────────────────────────────────────────────────┐
  │ Amplification factor G for Fourier mode θ:                   │
  │                                                              │
  │      1 - r sin²(θ/2)                                         │
  │ G = ──────────────────   where r = dt/dS²                    │
  │      1 + r sin²(θ/2)                                         │
  │                                                              │
  │ For all r > 0 and θ:  |G| ≤ 1                                │
  │                                                              │
  │ ⟹ UNCONDITIONALLY STABLE                                    │
  └──────────────────────────────────────────────────────────────┘

  No CFL Condition Required!
  • Can use any dt > 0
  • Can use any dS > 0
  • Grid spacing chosen for accuracy, not stability
