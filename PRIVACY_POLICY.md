**Encryption:**
- **In Transit:** TLS 1.3 for all data transmissions
- **At Rest:** AES-256 encryption for database and file storage
- **Hashed Passwords:** Bcrypt with 12 rounds (never stored in plaintext)
- **Encrypted Backups:** All database backups encrypted before storage

**Access Controls:**
- **Multi-Factor Authentication (MFA):** Available for all user accounts
- **Role-Based Access Control (RBAC):** Least privilege principle for staff access
- **API Authentication:** JWT tokens with 30-minute expiry, secure refresh mechanism
- **Session Management:** Automatic timeout after 15 minutes of inactivity

**Network Security:**
- **Firewall Protection:** Web Application Firewall (WAF) for DDoS and SQL injection prevention
- **Intrusion Detection:** Real-time monitoring for suspicious activity
- **IP Whitelisting:** Restricted administrative access
- **Geographic Restrictions:** Blocking access from high-risk countries

**Vulnerability Management:**
- **Regular Scanning:** Weekly automated vulnerability scans
- **Penetration Testing:** Annual third-party security audits
- **Patch Management:** Critical security patches applied within 48 hours
- **Dependency Monitoring:** Automated alerts for vulnerable libraries

### 9.2 Organizational Security Measures

**Staff Training:**
- Mandatory annual security awareness training
- GDPR and data protection training for all staff
- Incident response drills and tabletop exercises

**Access Restrictions:**
- Background checks for employees with data access
- Non-disclosure agreements (NDAs) for all staff and contractors
- Immediate access revocation upon termination

**Audit Logging:**
- Comprehensive logs of all data access and modifications
- Tamper-proof audit trails (immutable logs)

**Data Minimization:**
- Collect only data necessary for stated purposes
- Regular data cleanup and archival processes

### 9.3 Incident Response

- **To DPA:** Within 72 hours of becoming aware (GDPR Article 33)
